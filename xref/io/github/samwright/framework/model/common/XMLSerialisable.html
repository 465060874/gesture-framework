26      /**
27       * Creates a mutable clone of this object, with the data defined in the supplied XML
28       * {@link Node} where UUIDs in the node are first looked-up in the supplied dictionary.  If
29       * there is no match, the UUID is then looked-up in the {@link ModelLoader} class.
30       * <p/>
31       * When an object defined in the supplied {@code Node} is instantiated,
32       * the new object will be given a new UUID, and an entry will be made in the supplied
33       * dictionary to link from the old UUID and this new object.  Subsequent references to the old
34       * UUIDs are then dereferenced using the dictionary.  UUIDs of external objects (which
35       * are available to both the original and the new objects) are instead dereferenced in the
36       * {@code ModelLoader.getProcessor(uuid)} method.  Other references are assumed to be
37       * transient (eg. relating to controllers and will be recreated when needed) and thus ignored.
38       * <p/>
39       * It is therefore imperative that the new objects are created and the dictionary fully
40       * populated before any dereferencing of UUIDs is attempted.
41       *
42       * @param node       the XML node containing the data to populate the new object with.
43       * @param dictionary the dictionary to use for translating old UUIDs to new UUIDs.  In most
44       *                   cases, this should be an empty dictionary (which the function then
45       *                   populates).
46       */
47      XMLSerialisable withXML(Element node, Map<UUID, Processor> dictionary);

55      String getModelIdentifier();
