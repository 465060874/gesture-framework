3   /**
4    * An object which eventually becomes immutable.
5    * <p/>
6    * It might be immutable from creation, or it might start as mutable and become immutable.  Once
7    * immutable, it can not become mutable.
8    * <p/>
9    * The convention to use is {@code X getX()} for accessors and
10   * {@code EventuallyImmutable withX(newX)} for mutators.
11   * <p/>
12   * Whilst {@code this.isMutable()}, mutators mutate the internal data and return the same object.
13   * Once immutable, mutators create a mutable clone and mutations are applied to it, i.e.
14   * {@code createMutableClone().withX(newX)}.
15   * <p/>
16   * Mutations can therefore be strung together, e.g. {@code obj.withX(newX).withY(newY).withZ(newZ)}
17   * will return {@code obj} with the mutations if {@code obj.isMutable()} or
18   * {@code obj.createMutableClone()} with the mutations otherwise.
19   *
20   * @see io.github.samwright.framework.model.helper.MutabilityHelper MutabilityHelper
21   */
22  public interface EventuallyImmutable {
23      /**
24       * Create a mutable clone of this object.  It will remain mutable until passed as a
25       * {@code replacement} to {@code replaceWith(replacement)}, at which point
26       * {@code fixAsVersion(..)} will be called to effect the change to an immutable object.
27       *
28       * @return a mutable clone.
29       * @throws RuntimeException if this object is still mutable.
30       */
31      EventuallyImmutable createMutableClone();
32  
33      /**
34       * Returns true iff this object is mutable, meaning it was created as mutable and has not
35       * been finalised.
36       *
37       * @return true iff this object is mutable.
38       */
39      boolean isMutable();
